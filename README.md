### Dependencies
1. numpy
    * `pip3 install numpy`
2. textblob
    * `pip3 install textblob`
3. pytest
    * `pip3 install -U pytest`
4. requirements.txt (in GetOldTweetsPython)
    * `pip3 install -r src/GetOldTweetsPython/requirements.txt`
