### Dependencies
* numpy
    1. `pip3 install numpy`
* textblob
    1. `pip3 install textblob`
* pytest
    1. `pip3 install -U pytest`
* requirements.txt (in GetOldTweetsPython)
    1. `pip3 install -r src/GetOldTweetsPython/requirements.txt`
